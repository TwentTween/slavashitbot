# Slavashitbot 🤖

Telegram бот для воспроизведения аудио контента с категориями:
- 🎤 Про говно
- 🎵 Каверы  
- 💬 Цитаты

## Функции

- Воспроизведение аудиофайлов (.ogg)
- Интерактивное меню с кнопками
- Логирование всех действий
- Статистика использования
- Админские функции

## Команды

- `/start` - запуск бота
- `/admin` - админская панель (только для админов)
- `/stats` - статистика
- `/send <chat_id> <сообщение>` - отправить сообщение

## Деплой на Heroku

1. Создайте приложение на Heroku
2. Установите переменную окружения `BOT_TOKEN`
3. Подключите GitHub репозиторий
4. Включите worker dyno

## Локальный запуск

```bash
pip install -r requirements.txt
python main.py
```

## Переменные окружения

- `BOT_TOKEN` - токен Telegram бота
- `PORT` - порт для Heroku (автоматически)
